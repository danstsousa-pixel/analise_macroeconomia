# Dashboard Análise Macroeconomia

Um dashboard interativo desenvolvido em Python para análise de dados macroeconômicos e ativos financeiros em tempo real. 

Esta aplicação extrai, processa e visualiza indicadores de mercado, facilitando o acompanhamento de tendências e a tomada de decisão no cenário econômico.

## Acesse a Aplicação
O projeto está no ar e pode ser acessado diretamente pelo navegador:
**[(https://caprnhtgbuqn4bkx6ywx5r.streamlit.app/)]**

## Tecnologias Utilizadas
* **Python**
* **Streamlit:** Construção da interface web interativa.
* **Pandas:** Estruturação e manipulação de dados.
* **yfinance:** Extração de dados financeiros e de mercado em tempo real.
* **Plotly:** Geração de gráficos dinâmicos e interativos.

## Como executar o projeto localmente

### 1. Clone este repositório:
```bash
git clone [https://github.com/danstsousa-pixel/analise_macroeconomia.git](https://github.com/danstsousa-pixel/analise_macroeconomia.git)
```
### 2. Instale as bibliotecas necessárias:
```bash
  pip install -r requirements.txt
```
### 3. Execute a aplicação:
   ```bash
   streamlit run app_macro.py
   ```
